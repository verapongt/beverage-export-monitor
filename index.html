Recommended pipeline

  1. ตรวจสอบ endpoint หรือ CSV export บน customs.go.th
  2. ดึงตามเดือน ปี HS code และประเทศ พร้อม rate limit
  3. เก็บ raw response ก่อน transform เพื่อ audit
  4. validate เดือนซ้ำ ประเทศซ้ำ หน่วย และ missing value
  5. แสดง timestamp และ source URL ทุกครั้งใน dashboard

Methodology

นิยามและข้อจำกัด

Proxy definition

เริ่มจาก HS 2202 สำหรับน้ำ น้ำแร่ น้ำอัดลมที่เติมน้ำตาลหรือปรุงกลิ่นรส และเครื่องดื่มไม่มีแอลกอฮอล์อื่น ๆ ก่อนเจาะรหัสละเอียด 8 หรือ 11 หลักเมื่อมีข้อมูลจริง

Calculated metrics

MoM = เทียบเดือนก่อนหน้า, YoY = เทียบเดือนเดียวกันปีก่อน, Top 10 = ranking ตาม value_fob ในช่วงเวลาที่เลือก

Investor caveat

ข้อมูลนี้ช่วยอ่านภาพ macro ของตลาดส่งออก แต่ไม่ใช่คำแนะนำซื้อขายหุ้น และไม่ควรใช้แทนยอดขายบริษัทหรือ market share ระดับแบรนด์
